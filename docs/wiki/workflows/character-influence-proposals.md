# 角色影响提案：已由角色对话层替代

## Background

本页记录的“AI 生成多个候选方向、作者挑选其一”的方案已不再是产品方向。它会把角色降格为可配置的剧情选项，无法体现角色在对话中的抵触、误解、隐瞒和自主判断。

## Current Rule

新功能使用[角色对话层：以角色主体性承接作者意图](./character-dialogue-layer.md)。作者直接与角色自然交谈；角色基于思路线、硬事实、关系、资源与信息边界回应。对话形成的单条软性影响由作者决定是否带入后续创作，而非从多个剧情建议中选择。

旧的 `CharacterInfluenceProposal` 数据表保留在迁移历史中，以保证已部署数据库兼容；新工作台、运行时上下文和章节承接均不再消费它。不得在该表上新增产品行为。
